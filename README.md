# Deb Rosen Therapy Portfolio

Professional portfolio website for a Licensed Professional Counselor in Connecticut.

**Live site:** [debrosentherapy.com](https://www.debrosentherapy.com)

## About

A single-page therapy practice website featuring:

- Responsive design with mobile navigation
- Scroll-triggered animations
- Contact form with Netlify Forms integration
- SEO optimization with structured data
- Psychology Today verification badge

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Netlify (hosting & forms)

## Local Development

1. Clone the repository
2. Open `index.html` in a browser, or use a local server:
   ```bash
   npx serve
   ```

## Deployment

The site is deployed on Netlify with automatic deploys from the main branch.
